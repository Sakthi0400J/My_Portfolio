# My_Portfolio
## 🧠 Overview

This is a personal portfolio website designed with a **CLI / cyberpunk interface**, focusing on minimalism, interactivity, and clean UX.

The highlight is the **Handshake Protocol (Contact Form)** — a stylized communication system that simulates a secure uplink transmission.

---

## ✨ Features

- 🖥️ Terminal-style UI/UX
- 🎯 Minimal, responsive design
- ⚡ Smooth animations & transitions
- 🧾 Floating label input fields
- 📡 “Initiate Uplink” contact system
- 📩 Email delivery using FormSubmit
- 🌐 Fully static (no backend required)

---

## 🛠️ Tech Stack

- HTML5  
- CSS3 (Custom + Tailwind concepts)  
- JavaScript (Vanilla)  
- Form handling via FormSubmit  

---

## 📡 Contact System (Handshake Protocol)

The contact form uses:
### Flow:
1. User enters details  
2. Clicks **INITIATE UPLINK**  
3. Data is transmitted via FormSubmit  
4. Message is delivered directly to email
